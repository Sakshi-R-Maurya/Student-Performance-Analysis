# Student Performance Analysis

## Objective

Analyze student performance data using Python and perform data cleaning, analysis, and visualization.

## Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

## Tasks Performed

1. Loaded the dataset using Pandas.
2. Checked for missing values.
3. Removed duplicate records.
4. Analyzed student performance.
5. Calculated average final grades.
6. Studied correlation between study time and grades.
7. Compared performance based on gender.
8. Created visualizations:

   * Histogram of Grades
   * Scatter Plot (Study Time vs Grades)
   * Bar Chart (Gender vs Average Grade)

## Key Findings

* Average Final Grade (G3): 10.42
* Students scoring above 15: 40
* Study time showed weak positive correlation with grades.
* Male students had slightly higher average grades than female students.

## Conclusion

Student performance depends on multiple factors. Study time alone showed only a weak relationship with final grades in this dataset.
